pre.c:
A small c program that takes input in format "StudentName GPA". You can enter multiple entries and when finish you must CRTL-D. The program will show you which entries of students achieved a GPA of 3.0 and higher.

Compiling & Running:
gcc -o pre pre.c
./pre
